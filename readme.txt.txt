Requirements:
	1. at least i3 Inter processor
	2. 4GB of RAM
	3. Python version 3 Installed
	4. Libraries of python : pandas, matplotlib, streamlit, seaborn, urlextract, wordcloud, collections, emoji, re, nltk, keras, tensorflow installed.

After all the requirements are satisfied, run command "streamlit run app.py" in the directory where all the files are saved.
